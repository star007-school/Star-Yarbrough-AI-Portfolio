# Final Project - YOLOv8 Animal Detection System

## Problem
The goal of this project was to create a real-time computer vision system capable of detecting and identifying animals using deep learning techniques.

## Approach
I used YOLOv8 for object detection and tested the model using images and video data. The system was designed to recognize animals and display detections with bounding boxes and confidence scores.

## Results
The project successfully detected animals in images and video while demonstrating real-time object detection capabilities using modern AI techniques.

## Tools Used
- Python
- YOLOv8 (Ultralytics)
- OpenCV
- Google Colab

## Key Takeaway
This project demonstrated how modern object detection systems can be used in real-world AI applications such as animal recognition, monitoring, and automated visual analysis.
