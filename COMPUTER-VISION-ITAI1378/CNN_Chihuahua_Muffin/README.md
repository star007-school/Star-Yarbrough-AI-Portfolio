# CNN Image Classification - Chihuahua vs Muffin

## Problem
The goal of this project was to build a model that can correctly classify images as either a chihuahua or a muffin, which can be surprisingly difficult due to their visual similarities.

## Approach
I used a Convolutional Neural Network (CNN) to train on labeled image data. The model learns patterns such as shape, texture, and color to distinguish between the two classes.

## Results
The model was able to successfully classify most images and demonstrated how CNNs can extract meaningful features from visual data.

## Tools Used
- Python
- TensorFlow / Keras
- Google Colab

## Key Takeaway
This project helped me understand how CNNs process images and how feature detection works in deep learning.
