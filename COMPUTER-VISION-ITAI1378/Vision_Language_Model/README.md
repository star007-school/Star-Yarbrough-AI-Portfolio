# Vision-Language Model (CLIP / BLIP)

## Problem
The goal of this project was to explore how AI can understand both images and text together.

## Approach
I used a vision-language model to connect image features with text descriptions. This allowed the model to perform tasks like image classification and captioning without traditional training.

## Results
The model was able to match images with relevant text and generate meaningful descriptions, showing how multimodal AI works.

## Tools Used
- Python
- Hugging Face Transformers
- CLIP / BLIP models
- Google Colab

## Key Takeaway
This project showed how modern AI can combine vision and language, which is a big step beyond traditional models.
