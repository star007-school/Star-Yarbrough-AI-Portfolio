# Object Detection and Tracking (YOLO)

## Problem
The goal of this project was to detect and track objects in a video and analyze their movement.

## Approach
I used the YOLO model with tracking to detect objects in real time. A counting system was implemented to track objects as they crossed a defined zone.

## Results
The model successfully detected and tracked objects while ensuring each object was only counted once using unique tracking IDs.

## Tools Used
- Python
- YOLO (Ultralytics)
- OpenCV
- Google Colab

## Key Takeaway
This project demonstrated real-world computer vision applications like surveillance, traffic monitoring, and analytics.
